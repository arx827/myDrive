What is cypress

  • Test Automation Tool
  • Can test anything that runs in a web browser
  • Write tests in JavaScript or TypeScript
  • Does not use Selenium (todo: 補difference架構圖)
  • Open source

Supported browsers：https://docs.cypress.io/guides/guides/launching-browsers#Browsers

Features

  • Time Travel - takes snapshots as your tests run
  • Debuggability - readable errors and stack traces
  • Automatic Waiting - automatically waits for commands and assertions before moving on
  • Consistent Results - doesn’t use Selenium or WebDriver. Fast, consistent and reliable
  • Screenshots and Videos - get screenshots and videos
  • Cross browser Testing - locally or remote (CI CD)

See more features：https://docs.cypress.io/guides/overview/why-cypress#Features

Prerequisites

  • Windows 7 and above (64-bit only)
  • macOS 10.9 and above (64-bit only)
  • Linux Ubuntu 12.04 and above, Fedora 21 and Debian 8 (64-bit only)
  • Node.js 12 or 14 and above

the latest prerequisites here：https://docs.cypress.io/guides/getting-started/installing-cypress#System-requirements

Who uses Cypress

  • Developers or QA engineers building web applications using JavaScript frameworks
  • Cypress enables you to write all types of tests:
    • End-to-end tests
    • Component tests
    • Integration tests
    • Unit tests
